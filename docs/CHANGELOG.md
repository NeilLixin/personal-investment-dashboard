# Changelog

## v0.5.0 — 2026-07-01

- 导航结构产品化重构，主导航收口为八个日常入口。
- 持仓管理升级为持仓工作台，买卖计划和操作日志合并到持仓详情。
- 资产配置页面合并到总览、风险雷达和持仓设置。
- 数据同步、GitHub 同步和数据备份合并为“同步与备份”。
- 投资日报默认隐藏 Markdown 导出。
- 截图导入按平台+代码/规范化名称去重，默认覆盖已有持仓。
- 增加快速加仓、减仓、记录操作及按比例摊减成本。
- UI 统一中文、格式化金额收益率并隐藏技术字段。

## v0.4.0 — 2026-07-01

- 截图导入流程产品化简化，默认上传后自动 OCR 和自动解析。
- 新增东方财富五列持仓与账户摘要坐标解析。
- 普通用户默认不显示 OCR 调试信息，高级调试模式保留原文、预处理图和 items。
- 统一中文确认导入表格，继续要求人工确认后写库。
- 投资日报表格及 Markdown 中文化，数据同步页面简化。
- 风险雷达升级为仓位、收益、交易行为、复盘纪律、数据质量五维评分模型。

## v0.3.0 — 2026-07-01

- 增强支付宝基金持仓截图解析。
- 新增保留 box/中心坐标的 OCR items 结果。
- 新增移动截图裁剪、放大、轻度锐化、对比度增强和按行切分。
- 新增支付宝基金三列专用 parser、名称续行拼接、数字纠错和资产分类推断。
- 新增 OCR items 表格、解析诊断、可选调试图片和相关测试。
- 保留普通 OCR 与手动文本回退，所有结果继续人工确认后入库。

## v0.1.0 — 2026-06-29

- 初始化个人投资驾驶舱。
- 增加九个 Streamlit 页面、SQLite 六张表和自动初始化。
- 增加持仓计算、资产配置、Plotly 图表和本地风险雷达。
- 增加可选 RapidOCR、支付宝 parser、人工确认和 OCR 批次记录。
- 增加计划、操作日记、复盘、备份、demo、Git 同步和 pytest 测试。
- 补齐 README 与完整 docs 交接体系。
## v0.2.0 — 2026-06-30

- 新增结构化持仓数据跨设备同步。
- 新增投资日报和 Markdown 保存/下载。
- 增强风险规则、风险分和阈值配置。
- 增强交易字段、复盘统计和图表。
- 增强文档和测试。
- 修复 OCR 依赖缺失时按钮无解释地禁用；上传图片后按钮始终可操作并提供安装提示。
- 增加多图逐张 OCR、自动 OCR 开关、逐图结果、UploadedFile 复位和结构化错误返回。
- 增加 OCR 引擎与支付宝不完整识别文本测试。
