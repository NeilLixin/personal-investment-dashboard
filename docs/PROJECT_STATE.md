# Project State

## 当前状态

`personal-investment-dashboard` 是 Python + Streamlit + SQLite 的本地个人投资驾驶舱，当前版本为 v0.7.0。已具备持仓工作台、基金代码补全助手、截图 OCR 导入、今日市场快照、投资日报、五维风险雷达、复盘统计，以及 Mac/Windows 结构化同步和本地备份能力。

主导航已收口为八个入口：总览、持仓工作台、截图导入、投资日报、风险雷达、复盘中心、同步与备份、设置。计划、交易记录和持仓设置已集中到持仓工作台；资产配置和同步相关页面也已完成合并。

## 当前主要问题

- 支付宝和东方财富页面变化可能影响坐标解析，仍需更多脱敏样本覆盖。
- 部分页面仍有工程化痕迹，空状态、错误提示和高级信息隐藏可继续优化。
- 风险阈值和复盘洞察需要真实长期使用后校准。
- 暂无价格快照历史、复杂绩效归因、多用户和公网部署能力。
- AKShare 是可选数据源；接口或网络失败时需要使用截图/手动文本，场外基金当天净值可能尚未更新。
- 市场 API 依赖有效的 6 位代码；缺失时在“持仓工作台 → 基金代码补全助手”刷新候选库、保守匹配并人工确认。截断名称、A/C 歧义和多候选不会静默写库。

## 下一步

优先稳定收益截图的更多 App 版式、验证 AKShare 接口兼容性、打磨持仓工作台和同步流程，并持续减少普通用户可见的技术字段。

## 当前不做

不进行 React 重写，不接券商或自动交易，不预测行情，不承诺收益，不把应用暴露到公网。
